# Kagglib
- Library for Kaggle competitions

## Usage
### nlp
For neural language processing(nlp) tasks
- adversarial validation
    - awp, fgm
- preprocessing
    - fix encoding
- model
    - custom bert model example
    - reinit bert
- optimizer
    - get scheduers

### nn
For neural network(nn) tasks
- model
    - pooling
    - mixout
    - freeze
- utils
    - average meter
    - early stopping

### tabular
tabular tasks
- blocks
    - wrapper
    - aggregation
    - encoding
- model
    - gradient boosting decision tree
        - lightgbm, xgboost, catboost

### utils
utility functions
- exp_manage
    - set wandb
- utils
    - Timer
    - reduce memory usage
    - seed everything
    - get logger
